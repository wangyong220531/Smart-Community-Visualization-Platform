# 智慧社区可视化平台

## 项目启动

```shell
yarn
yarn dev
```

## 项目描述

一个智慧可视化小区项目（纯前端）

## 技术栈

React Hooks + React Router + Zustand + Less + Canvas + Axios + Antd
